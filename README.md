## 🚀 How to Use

### Opening Locally
1. Make sure you have [Node.js](https://nodejs.org/) installed
2. Navigate to the project folder
3. Run `npm install` to install all dependencies
4. Run `npm run dev` to start the development server
5. Open [http://localhost:8080](http://localhost:8080) in your browser

### Tips for the Best Experience
- Open it on a quiet evening, put on some headphones 🎧
- Turn on the music (click the 🎶 button in the bottom right)
- Scroll slowly and take time to read everything
- Click on things – there are surprises hidden throughout
- Visit the secret page at `/papoyeee` for something extra special
- Try the Konami code if you're feeling adventurous 🎮

## 🎨 Customization

Want to personalize this further? Here's what you can easily change:

### Colors
Update the CSS variables in [src/index.css](src/index.css) to your favorite shades.

### Content
All the text about you, our story, and memories can be updated in the component files:
- `src/components/AboutSection.tsx` - Your info
- `src/components/LoveStorySection.tsx` - Our timeline
- `src/components/PromisesSection.tsx` - My promises
- `src/components/LoveLetterSection.tsx` - Love letter

### Photos & Videos
Replace the images in `src/assets/` with your favorite memories of us together.

### Music
Replace `public/sounds/our-song.mp3` with our favorite song.
